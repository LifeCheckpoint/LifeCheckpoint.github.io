『不止的思念，互相的救赎』：《樱花、萌放》长评与感念

本人已成为小黑厨（笑，Kuro 我喜欢你❤），本篇长评（以及不时出现的奇怪幻想）主要围绕小黑展开...所以其它角色的剧情过的偏快一些，如果有什么理解漏洞，请尽情批评，共同讨论改进~

在结束之后，在一切之前​

当我关闭樱花萌放的界面，听着 《輪廻》 这首 OP 的时候，心潮澎湃，似乎来到那辆永远奔跑在夜之国时空的列车上，与小黑相枕而眠。

回想起自己打开这部作品中的原因，大概是被群友可爱的表情包吸引了吧~


万恶之源.gif

随后一路来到萌百，看了小黑的角色简介后，决定体验这部作品 （嗯，喜欢笨手笨脚的萝莉） 。下载游戏的时候被惊讶了一下，体积非常大，也许有很多的语音和 CG，事实证明确实如此——将近百小时的慢热剧情，彻底满足了我对这个世界观窥探的愿望，也培育起了对里面角色的衷心喜爱。

后续就是将近大半个月的推进过程，平时在学校也比较忙，只能趁着那些第二天没有早八的夜晚，翻阅着一篇又一篇动人心弦的故事。

不知不觉耳边的歌声已经停止，趁着心中的激动尚未平复，打算写一篇评论与感想，讨论一些作品的内容。

《樱花，萌放》总体评价​

《樱花萌放～as the Night’s，Reincarnation》，是一部 2019 的作品。这部作品在批评空间取得了 90 分的优秀成绩。对我而言，值——脚本写作者漆原雪人的构思，叙事与文笔都很能对上电波，因此也逐渐变成忠诚的 fp（乐）。

坏话说在前头（）这部作品的 缺点 （或者说，不那么容易被广泛接受的特点）是...


雪人语（雪人写作特有的啰嗦）​

有一段经典的雪人语叙述——

听好了，十夜，好好听着哦

嗯…听着呢，从刚才开始就一直好好听着呢

嗯，是啊，十夜的话肯定没问题的吧…虽然我相信你是没问题的…但是我还是希望你能听听这件事

嗯，嗯…是什么？

那个啊，十夜。听好了，十夜…

有很多同好们游玩的时候可谓是对这种啰嗦 深痛恶绝，哈哈，毕竟一段啰嗦的文本总是会给人烦躁的感觉，然后按下 ctrl 一路狂飙（希望正在读这段文字的各位手下留情）

不过相反，我有点 喜欢 这种风格。首先引用贴吧老哥的一段评论：

是的，雪人说话确实啰嗦，但是，当我真的沉下心去阅读每一句台词的时候，总能从重复的几句话中体味出不一样的味道，以刚才与那位“狂妄的幼女”十夜的对话为例：

...

（第一次叙述，没有特别的感觉，只是知道接下来要说的话有些重要）

（是的，我一直相信着十夜，但我似乎已经进入了一种沉浸的状态，祈求着十夜能听听自己说话）

（我近乎央求地说着这些，感觉从一个忠告者变成了祈求者）

...

所以，细细品读，每一句的情感风味还是有所不同，如果能多给自己一点点耐心，咀嚼这些看似重复啰嗦的话语，私以为能体味出文字的力量。

话说回来，作为一个商业作品，写太多废话也并不喜人。如果真的有点厌烦的话，暂停游戏，让我们过段时间再继续推进。

超长剧情​

正如前面所说的，剧情也太长了...自然，这是这部作品的 缺点 之一。没日没夜地推进，仅仅是共通线已经是两三部其它 Gal 的短剧情长度了（雪人语功不可没），花费时间太久，经常需要中断，而中断后又会有些迷茫，要不要打开游戏呢？我就因为工作日只有晚上数个小时的时间而为推进剧情感到窘迫——都要到最后的小黑线剧情了，情感流如果为时间所迫断开的话，就不会有如此震撼的感觉了...就这一点来说，很难受，如果游玩，最好还是找一段假期慢慢地体味...

下面是剧情流程图，扫一眼就知道有多复杂了

不过，长剧情也是这部作品的 优点 之一。长，意味着慢热，意味着我有足够的时间去融入和体会作者的想法，体会剧中百态，甚至去爱上剧中的某个角色（小黑，你说是吧🤤），而非一路速通，最后只是怅然若失地，毫不满意地吐槽“没看够哇！”（点名批评某匿名废萌作品《家喵二三事》）

虐女主有些过分​

可能各位对“虐”的定义有些不同， 就我个人而言，我有一些难接受剧情中发刀子的片段。印象最深刻的片段，是小春线中，大雅（我）隐瞒了我将要通过在当前时间线中饮弹自尽以求穿越到过去拯救小春的事实，拿到了最看重大雅的生命的小黑的“魔法许可”，从而造出了回到过去的子弹。当小黑授予许可的时候，立刻知道了这个事实，便开始大哭，最后大雅趁所有人都不在的时刻，结束了在这个世界线他的生命。

看这一段的时候，心情五味杂陈，虽然后文有一些解释的情节试图圆回这一段，但当我与小黑共情之时，就由衷地感到痛苦与无力。无法想象一个希望为了大雅的幸福献出自己全部的小女孩，得知她深爱的人，要利用与她共同缔造的——本应为幸福而发挥的魔法——自尽时，究竟感到多么痛苦、无助和自责。所以我和小黑是一起哭的，甚至想骂一骂漆原雪人那种“好人就该被自己拿枪指着”的逆天写法偏好。

后文还有很多相关内容，放到下文详细部分说明。平心而论，为了剧情的完整性、指向性，以及为了情绪的渲染，有些“虐”可能确实不能避免，但是在这一点上我依然无法原谅作者的处理...

另外有一些说法认为，雪人把他曾遭受的精神痛苦都投射到了他的作品中，所以有一种扭曲的黑暗倾向...嗯，我是认同的，实在可恶。

回到作品上来，这部作品的优点也是显而易见的


优秀的童话感​

雪人笔下的故事，都有着无与伦比的 童话感。

一直以来我有一个误区，认为童话是分为面向成年人与儿童的。但是在查阅了一些资料后，才意识到，童话并没有成人和儿童之分。一些创作者认为...

原创作品，也许可以称为“拟童话”。

本身针对的读者是成人，但碰巧具有符合儿童心理的成分，为儿童所爱读，因此被误解为童话，其实它不是。而且它的内容不一定适合儿童，儿童也不一定能够理解。

例子：《格列佛游记》、《动物庄园》、《西游记》、《呼兰河传》。

对于樱花摸鱼这部作品而言，我认为它的童话感主要凸显于下面两个部分：

  • 本就强依赖于儿童经验的世界观：当人们真正地长大（作为成熟的标志），他们会忘记与“夜”有关的一切，而后者正是世界观构建的最核心基础。
  • 大量以儿童为视角基础输出的叙事：作品中出现的主要人物，大多数都仔细地描绘了他们的童年。“我”作为孩子，“我”这样想，“我”这样言说，我所做的行动的谋划都基于我尚不成熟（真的不成熟吗？）的心灵
  • 强大的奇幻叙事：摸鱼的叙事奇幻程度很高，作者对于故事时间与空间的把握强悍，因此支撑住了这个庞大的，由想象化形的有关“夜”的叙事。

然而，童话感并不一定会带来幸福感，或是一定有什么圆满的结局，它终归只是一些从严酷的事实中所组织起来的诗化的华彩。同时，由雪人投射到文本中的那种痛苦，在儿童身上被表露后，其效果是更为黑暗的。

总的而言，童话感的渗透让这部作品的叙事得以展现出它最奇幻的那一面。在下面的详细介绍中，我会尝试寻找这些闪光点。
