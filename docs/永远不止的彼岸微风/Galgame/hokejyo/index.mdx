## 世界观

<ProgressBar progress={55} /> **5.5 / 10**

### 魂人

世界观的核心，是 **魂人** 。如果一个人在死前有着很深的执念，无论是爱、不舍、遗憾还是怨恨，都会变成一种介于生存与彻底死亡之间的中间状态。

一旦魂人实现了自己的执念，他便可以选择 **归还** 天空，不再留下遗憾。

魂人拥有 **灵力** ，可以通过灵力发动 **奇迹** ，每个人的奇迹都与自己的执念相关，独一无二。而作为消耗灵力的原料，他们同样需要付出相应的 **代价** 。

例如，

|人物|奇迹|代价|
|---|---|---|
|白花|自由操纵风的能力|灵力完全消耗将无法行走|
|乙姬|抹消现象，将一切现象归于“无”的能力|与性相关的物品会被消耗|

乍看上去似乎代价有些无厘头，然而，代价与自己生前执念 **息息相关** ，是魂人执念的具象化表征之一。

魂人保持着死亡前的年龄，认知水平与死前相似，可能有缓慢的增长。

### 送归人

总的来说，送归人 **化解** 魂人的执念，并使魂人归还天空。

送归人的流派也比较多样，既分为倾向于强制手段的、非强制手段的，也有採物（送归工具）之别。在这当中，主角 **风见苍空** 归属无採派——一种最特殊的流派，他们不使用採物进行送归，而仅凭借 **言灵** ，通俗来说也就是基于语言的心理治疗，进行送归。

### 世界观总结

大体合理的奇幻学框架，“魂人”的设定完全支撑起了这个故事，然而细节经不起推敲。

1. 魂人的能力差别较大，在如何保持世界的整体平衡方面，自圆其说存在困难。
2. 叙事长达五六十年的跨度，魂人与人类一同生活，其存在的事实却能被长期掩盖，不算非常合理

基础分 5，世界观支撑力强，+0.5。

## 叙事情节

<ProgressBar progress={20} /> **寻找姐姐 2 / 10**

<ProgressBar progress={85} /> **小白身世 8.5 / 10**

<ProgressBar progress={70} /> **HS 部分 7 / 10**

作品 **多线串联** ，在篇幅上显著分为了多个部分：

 - 寻找变为魂人的姐姐，以及
 - 小白&主角的身世回忆

### 寻找姐姐

主角苍空的姐姐为了照顾他，早衰而死，成为魂人。苍空为了亲手送归姐姐，选择成为一名送归人。

苍空与小白一同来到彩香镇，据说这里有大量魂人的聚集，找到姐姐的概率更高。为此，苍空成为了一名保健老师，而白花成为了他的助手。

~~名义助手，大部分时间其实在添乱\~~~

这一部分，如果直说，写得实际上较差，叙事混乱，常常有不知所谓的突然剧情插入，除了小白以外的角色描写都常常不入状态。

叙事还存在着一个问题，前期节奏过于平缓，偶有小高潮也与主线关系偏弱，后文却猛踩油门，让人有些不知所措。

这也许是很多同题材 Gal 的通病，即使是剧情优秀的《星空列车与白的旅行》同样无法避免这个问题。

最终决定给 2 分。

### 小白身世

后半作文笔逐渐在线，笔力非常大。

这一部分的叙事是 **多时间线的交替** ，既在讲述 1981 年苍空与小白的故事，也讲述着 2020 年苍空与小白的羁绊。

### HS

HS 本身的想法和作画都不错，然而有一部分 HS 的插入与主线事件背离，像是强行加塞，叙事上有提升空间，给 7 分。

其实令人比较惊讶的是，白花的内心独白，以及面对归还的无比勇气，却在一个与苍空相拥而睡的 HS 中表现了，本应是沉浸于牝牡之乐的场景，却被用来呈现一段内心独白，呈现痛苦，流泪与微笑。
从这之中，可以看出写者的一部分意图，将严肃的自我升华放在常被认为是用于满足欲望的片段，创造的不协调将反过来促成读者对爱的本质的反思。

### 创新叙事：打破性别与年龄的枷锁

**性别** 是 Galgame 叙事的核心，任何男性向的游戏不会离开这个论域。
男性-女性的二元结构为 Galgame 提供了其 **恋爱冒险内核** 的根本框架。然而这一框架在便利其叙事文本提供给其受众满足感的同时，也在遮蔽着读者从不同视角去看待其哲学核心的双眼。
因此，我们会思考，如何在这一框架的基础下，尝试去进行变易，提供更多新颖的视角呢？也许这就是《保健室的老师与沉迷吹泡泡的助手》这部作品的创新叙事动机之一。

在剧情的后期，写者终于向读者揭示了主角苍空的前世，此花苍的性别， **女性** 。

回顾剧情，此花苍与白花本是同学，因为一系列契机而互相爱上，为了实现重病的白花看看海和风车的梦想，此花苍将白花艰难地背上山，愿望得以实现。
然而，白花的一个意外行动让她跌落悬崖，此花苍没能成功挽救，两人的生命最终结束在悬崖下的蒲公英田。在将近 40 年后，魂人白花才终于寻找到转生的苍空（此花苍）。

在历史线的发展过程中，最直接呈现的，便是两个女孩发生了相爱。然而，与一般 Galgame 的男-女，或是女-女的爱情故事有所不同，由于 **转生** 这一机制的存在，主角发生了性别的倒转——由女性变为男性。
在此，故事超越了一般的性别秩序，实际上剥离了对性别话题的拷问，而直指核心话题： **爱与什么有关？**
这种叙述为受述者玩家提供了身份上的张力，进而追问自己：
*我对爱的认同从何而来？是男性-女性二元结构让我产生了这种认同，还是性别之下，一些被遮蔽的其它因素？*

这种创新而平衡的叙事策略，文学上较为接近 **温和的女性主义**

这里还要补充的一点是，此花苍的女孩身份，是由白花亲口说出的，在这里 Galgame 的超文本叙事得到了强烈的印证：
白花真的说过这句话吗？这是白花的亲口吐露，还是苍空对他被激起的回忆的主动塑造，还是无关任何角色——仅仅是写者想要告诉读者的信息？就我而言，也许是第一、第二者的融合。

在此，真实与虚幻交织在一起，拓展了叙事的张力。

**年龄** 相关的论述，总体来说是敏感的。当然可以说发行商瞄准的目标受众是萝莉控 ~~（吃子弹吧！）~~ ，然而这里我也提供一个简要的，稍有不同的解释。

我们常说 **童言无忌** 。为什么呢？作为儿童身份，于社会上没有复杂的关系，不需要复杂的利害关系思虑，认知上暂时达不到处理复杂事务的水平，因此说话不会忌口。

然而，在本作中采用了以下叙事手段：

 - 大量情节的展开充实，都放在了前世主角与未亡女主的学生（孩童）时段
 - 认知水平依然是孩子的魂人白花，以其孩子气的口吻，发出与其年龄特征不相符的痛苦的自白

这实际上是一种 **塑造反差** 的常见手法，在文学作品中较为常见,在本作以至同类作品中都有类似片段。
白花处于孩子的认知水平，但是她依旧经历了五十多年的生活经历，这些经历让她能以不相称的身份，对苍空叙说羁绊、索求与痛苦，反差十分强烈。

在故事高潮，白花已经实现了她的执念——让苍空变得幸福，然而与苍空的数年旅行已经让她不想分别。
由此，她要做出是否归还天空的抉择。夜深人静的床上，白花流着泪进行着独白：

## 思想内核

<ProgressBar progress={65} /> **6.5 / 10**

经典的 **别离 / 生死观** 话题，少部分话题还是会落入俗套，故基础分 6。然而世界观的加入能够为这个常见的哲学话题提供一些变数：现象学中的“向死而生”阐述了人如何将死亡作为本质焦虑而推动日常的生活，剧情则提出“向生而死”，反弹琵琶，其关键论题为“为了生者，我们应该如何做出死亡的选择”，也即“如果自己可以在死亡前做出抉择，人的选择将会如何变化”。剧情中“魂人”就是其具象化的表现。
