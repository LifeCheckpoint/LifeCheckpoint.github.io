01. 逻辑与证明

1. 命题逻辑 (Propositional Logic)​

1.1 命题与逻辑联结词​

  • 命题 (Proposition)：一个可以判断真假的陈述句。

    • 真值 (Truth Value) 只能是 真 (True, T) 或 假 (False, F)。
    • 例如：1 + 1 = 2 (T)，深圳在夏天 (F)。
    • 疑问句、祈使句、或含有不确定变量的句子不是命题。

  • 复合命题 (Compound Proposition)：由一个或多个简单命题通过逻辑联结词 (Logical Connectives) 组合而成。

  • 核心逻辑联结词

或/析取 (Disjunction)orp∨qp \lor qp∨q“p 或 q”当 ppp 和 qqq 至少一个为真时，结果为真。

1.2 深入理解蕴含 (p→qp \to qp→q)​

蕴含是逻辑中最关键也最易混淆的概念之一。

  • 定义: p→qp \to qp→q 为假，当且仅当前提 ppp 为真，结论 qqq 为假。
  • 核心思想: 蕴含代表一种“承诺”。只有当“前提条件达到，但承诺未兑现”时，这个蕴含命题才是假的。
  • 用例: “如果你期末考了 100 分 (ppp)，那么你的总成绩就是 A (qqq)。”
    • 你考了 100 分 (T)，但没拿到 A (F)   ⟹  \implies⟹ 老师说谎了 (p→qp \to qp→q 为 F)。
    • 你没考 100 分 (F)，无论你是否拿到 A (T/F)   ⟹  \implies⟹ 老师没有违背承诺 (p→qp \to qp→q 为 T)。这种情况称为空虚的真 (Vacuously True)。
  • 重要等价形式: p→q≡¬p∨q‾\underline{p \to q \equiv \neg p \lor q}p→q≡¬p∨q​。这个转换在证明中极为常用。

1.3 逆命题、否命题与逆否命题​

对于一个蕴含式 p→qp \to qp→q：

类型英文形式与原命题关系
原命题 (Implication)p→qp \to qp→q-
逆命题 (Converse)q→pq \to pq→p不等价
否命题 (Inverse)¬p→¬q¬p \to ¬q¬p→¬q不等价
逆否命题 (Contrapositive)¬q→¬p‾\underline{¬q \to ¬p}¬q→¬p​逻辑等价

考点: 证明 p→qp \to qp→q 等价于证明它的逆否命题 ¬q→¬p¬q \to ¬p¬q→¬p。

2. 逻辑等价 (Logical Equivalence)​

  • 恒真式 (Tautology): 无论其包含的命题变量取何值，永远为真的复合命题 (例如 p∨¬pp \lor \neg pp∨¬p)。
  • 矛盾式 (Contradiction): 永远为假的复合命题 (例如 p∧¬pp \land \neg pp∧¬p)。
  • 逻辑等价 (Logically Equivalent): 如果 p↔qp \leftrightarrow qp↔q 是一个恒真式，那么称 ppp 和 qqq 是逻辑等价的，记为 p≡qp \equiv qp≡q 或 p⇔qp \Leftrightarrow qp⇔q。

3. 谓词逻辑 (Predicate Logic)​

3.1 谓词与量词​

命题逻辑无法表达“所有”、“存在”等概念，谓词逻辑通过引入变量和量词解决了这个问题。

  • 谓词 (Predicate): 一个带变量的陈述，当变量被具体值替换后，变成一个命题。记为 P(x),Q(x,y)P(x), Q(x, y)P(x),Q(x,y) 等。
    • 论域 (Universe of Discourse): 变量可以取值的集合。
  • 量词 (Quantifier): 用来描述变量范围的符号。
名称英文符号含义如何为真如何为假
全称量词 (Universal)For all∀∀∀“对于所有/任意”论域中所有 xxx 都使 P(x)P(x)P(x) 为真。存在一个 xxx 使 P(x)P(x)P(x) 为假。
存在量词 (Existential)There exists∃∃∃“存在/有些”论域中至少有一个 xxx 使 P(x)P(x)P(x) 为真。论域中所有 xxx 都使 P(x)P(x)P(x) 为假。

3.2 自然语言翻译 (重点)​

将自然语言翻译成谓词逻辑表达式是常考题型，需注意蕴含和合取的正确使用。

  • “所有 P 都是 Q”: 应翻译为 ∀x(P(x)→Q(x))∀x(P(x) \to Q(x))∀x(P(x)→Q(x))。
    • 用例: “所有学生都很聪明。” (论域：所有人)
      • 正确: ∀x(Student(x)→Smart(x))∀x(\text{Student}(x) \to \text{Smart}(x))∀x(Student(x)→Smart(x)) (如果一个人是学生，那么他很聪明)
      • 错误: ∀x(Student(x)∧Smart(x))∀x(\text{Student}(x) \land \text{Smart}(x))∀x(Student(x)∧Smart(x)) (这意味着所有人既是学生又很聪明)
  • “有些 P 是 Q”: 应翻译为 ∃x(P(x)∧Q(x))∃x(P(x) \land Q(x))∃x(P(x)∧Q(x))。
    • 用例: “有些学生很聪明。” (论域：所有人)
      • 正确: ∃x(Student(x)∧Smart(x))∃x(\text{Student}(x) \land \text{Smart}(x))∃x(Student(x)∧Smart(x)) (存在一个人，他既是学生又很聪明)
      • 错误: ∃x(Student(x)→Smart(x))∃x(\text{Student}(x) \to \text{Smart}(x))∃x(Student(x)→Smart(x)) (这个命题的意义很弱，只要存在一个不是学生的人，它就为真)

3.3 量词的否定 (德摩根定律)​

对量化命题的否定是另一个高频考点。

  • ¬∀xP(x)≡∃x¬P(x)¬∀x P(x) \equiv ∃x ¬P(x)¬∀xP(x)≡∃x¬P(x)
    • “并非所有人都聪明” ≡\equiv≡ “存在不聪明的人”
  • ¬∃xP(x)≡∀x¬P(x)¬∃x P(x) \equiv ∀x ¬P(x)¬∃xP(x)≡∀x¬P(x)
    • “不存在完美的人” ≡\equiv≡ “所有人都不完美”

3.4 嵌套量词​

当多个量词同时出现时，它们的顺序非常重要。

  • 同类量词: 顺序可交换。
    • ∀x∀yP(x,y)≡∀y∀xP(x,y)∀x∀y P(x, y) \equiv ∀y∀x P(x, y)∀x∀yP(x,y)≡∀y∀xP(x,y)
  • 不同类量词: 顺序不可交换，含义截然不同。
    • 用例: 论域为所有人，L(x,y)L(x, y)L(x,y) 表示 "xxx 爱 yyy"。
      • ∀x∃yL(x,y)∀x∃y L(x, y)∀x∃yL(x,y): 每个人都爱着某个人。(爱慕对象可以不同)
      • ∃y∀xL(x,y)∃y∀x L(x, y)∃y∀xL(x,y): 存在一个人，被所有人爱着。(万人迷)

4. 证明 (Proofs)​

4.1 证明的术语与推理规则​

  • 公理 (Axiom): 不证自明的基本命题。
  • 定理 (Theorem): 可以被证明为真的命题。
  • 证明 (Proof): 建立一个定理为真的有效论证过程。
  • 推理规则 (Rules of Inference): 从一组真前提推导出真结论的有效逻辑步骤。
规则名称英文形式含义
假言推理 (Modus Ponens)Affirming the antecedentp→q, p∴q\frac{p \to q, \ p}{\therefore q}∴qp→q, p​如果 ppp 蕴含 qqq，且 ppp 为真，则 qqq 为真。
拒取式 (Modus Tollens)Denying the consequentp→q, ¬q∴¬p\frac{p \to q, \ \neg q}{\therefore \neg p}∴¬pp→q, ¬q​如果 ppp 蕴含 qqq，且 qqq 为假，则 ppp 为假。
假言三段论 (Hypothetical Syllogism)p→q, q→r∴p→r\frac{p \to q, \ q \to r}{\therefore p \to r}∴p→rp→q, q→r​蕴含关系具有传递性。
析取三段论 (Disjunctive Syllogism)p∨q, ¬p∴q\frac{p \lor q, \ \neg p}{\therefore q}∴qp∨q, ¬p​如果 ppp 或 qqq 为真，且 ppp 为假，则 qqq 必为真。
简化律 (Simplification)p∧q∴p\frac{p \land q}{\therefore p}∴pp∧q​如果 ppp 和 qqq 都为真，则 ppp 为真。
合取律 (Conjunction)p, q∴p∧q\frac{p, \ q}{\therefore p \land q}∴p∧qp, q​如果 ppp 和 qqq 分别为真，则它们的合取也为真。
附加律 (Addition)p∴p∨q\frac{p}{\therefore p \lor q}∴p∨qp​如果 ppp 为真，则 ppp 或任何其他命题 qqq 都为真。

4.2 核心证明方法​

以下是非形式化证明中常用的策略，是解决证明题的关键。

  1. 直接证明 (Direct Proof)

    • 策略: 证明 p→qp \to qp→q。从假设 ppp 为真开始，通过一系列逻辑推导，最终得出 qqq 为真。
    • 用例: 证明“如果 nnn 是奇数，那么 n2n^2n2 也是奇数”。
      • 证明: 假设 nnn 是奇数，则 n=2k+1n = 2k + 1n=2k+1 (其中 kkk 为整数)。
      • 那么 n2=(2k+1)2=4k2+4k+1=2(2k2+2k)+1n^2 = (2k+1)^2 = 4k^2 + 4k + 1 = 2(2k^2+2k) + 1n2=(2k+1)2=4k2+4k+1=2(2k2+2k)+1。
      • 由于 2k2+2k2k^2+2k2k2+2k 是整数，所以 n2n^2n2 的形式是 2×(整数)+12 \times (\text{整数}) + 12×(整数)+1，因此 n2n^2n2 是奇数。

  2. 逆否证明 (Proof by Contrapositive)

    • 策略: 证明 p→qp \to qp→q。通过证明其等价的逆否命题 ¬q→¬p¬q \to ¬p¬q→¬p 为真。
    • 用例: 证明“如果 3n+23n+23n+2 是奇数，那么 nnn 是奇数”。
      • 证明: 证明其逆否命题：“如果 nnn 不是奇数（即偶数），那么 3n+23n+23n+2 也不是奇数（即偶数）”。
      • 假设 nnn 是偶数，则 n=2kn = 2kn=2k (其中 kkk 为整数)。
      • 那么 3n+2=3(2k)+2=6k+2=2(3k+1)3n+2 = 3(2k)+2 = 6k+2 = 2(3k+1)3n+2=3(2k)+2=6k+2=2(3k+1)。
      • 由于 3k+13k+13k+1 是整数，所以 3n+23n+23n+2 是偶数。逆否命题成立，故原命题成立。

  3. 反证法 (Proof by Contradiction)

    • 策略: 证明命题 ppp。先假设 ¬p¬p¬p 为真，然后从这个假设出发，推导出一个矛盾（例如 q∧¬qq \land ¬qq∧¬q 或与已知公理/定理相悖的结论）。这说明假设 ¬p¬p¬p 是错误的，因此 ppp 必为真。
    • 用例: 证明“2\sqrt{2}2​ 是无理数”。
      • 证明: 假设 2\sqrt{2}2​ 是有理数。
      • 那么存在互质的整数 m,nm, nm,n 使得 2=m/n\sqrt{2} = m/n2​=m/n。
      • 平方得 2=m2/n22 = m^2/n^22=m2/n2，即 m2=2n2m^2 = 2n^2m2=2n2。这说明 m2m^2m2 是偶数，从而 mmm 也是偶数。
      • 设 m=2km = 2km=2k，代入得 (2k)2=2n2  ⟹  4k2=2n2  ⟹  n2=2k2(2k)^2 = 2n^2 \implies 4k^2 = 2n^2 \implies n^2 = 2k^2(2k)2=2n2⟹4k2=2n2⟹n2=2k2。这说明 n2n^2n2 是偶数，从而 nnn 也是偶数。
      • mmm 和 nnn 都是偶数，这与它们互质的假设相矛盾。
      • 因此，最初的假设“2\sqrt{2}2​ 是有理数”是错误的。故 2\sqrt{2}2​ 是无理数。

  4. 分情况证明 (Proof by Cases)

    • 策略: 证明 (p1∨p2∨⋯∨pn)→q(p_1 \lor p_2 \lor \dots \lor p_n) \to q(p1​∨p2​∨⋯∨pn​)→q。分别证明 p1→qp_1 \to qp1​→q, p2→qp_2 \to qp2​→q, ..., pn→qp_n \to qpn​→q。如果所有情况都能导出结论 qqq，则原命题成立。
    • 用例: 证明“对于任意实数 x,yx, yx,y，有 ∣x∣∣y∣=∣xy∣|x||y| = |xy|∣x∣∣y∣=∣xy∣”。
      • 证明: 分四种情况讨论 x,yx, yx,y 的符号：
        1. x≥0,y≥0x \ge 0, y \ge 0x≥0,y≥0: ∣x∣=x,∣y∣=y,xy≥0  ⟹  ∣xy∣=xy|x|=x, |y|=y, xy \ge 0 \implies |xy|=xy∣x∣=x,∣y∣=y,xy≥0⟹∣xy∣=xy。xy=xyxy = xyxy=xy 成立。
        2. x≥0,y<0x \ge 0, y < 0x≥0,y<0: ∣x∣=x,∣y∣=−y,xy≤0  ⟹  ∣xy∣=−(xy)|x|=x, |y|=-y, xy \le 0 \implies |xy|=-(xy)∣x∣=x,∣y∣=−y,xy≤0⟹∣xy∣=−(xy)。x(−y)=−xyx(-y) = -xyx(−y)=−xy 成立。
        3. x<0,y≥0x < 0, y \ge 0x<0,y≥0: ...
        4. x<0,y<0x < 0, y < 0x<0,y<0: ...
      • 所有情况均成立，故原命题得证。
