02. 集合与函数

1. 集合的基本概念​

1.1 定义与表示​

  • 集合 (Set)：一个无序的对象集合。集合中的对象称为元素 (element) 或成员 (member)。
  • 集合相等：两个集合 AAA 和 BBB 相等，当且仅当它们拥有完全相同的元素，记为 A=B  ⟺  ∀x(x∈A↔x∈B)A = B \iff \forall x (x \in A \leftrightarrow x \in B)A=B⟺∀x(x∈A↔x∈B)。
  • 集合的表示方法：
    • 列举法：S={2,3,5,7}S = \{2, 3, 5, 7\}S={2,3,5,7}
    • 省略号法：A={1,2,3,...,100}A = \{1, 2, 3, ..., 100\}A={1,2,3,...,100}
    • 构造法 (Set builder)：{x∣P(x)}\{ x \mid P(x) \}{x∣P(x)}，表示所有具有性质 PPP 的元素 xxx 的集合。

1.3 特殊集合与子集​

  • 全集 (Universal set): 在特定上下文中，所有讨论对象构成的集合，记为 UUU。
  • 空集 (Empty set): 不包含任何元素的集合，记为 ∅\emptyset∅ 或 {}\{ \}{}。
    • 常考易混点：空集本身是一个集合，而包含空集的集合不是空集。即 ∅≠{∅}\emptyset \neq \{\emptyset\}∅={∅}，因为后者包含一个元素：∅\emptyset∅。
  • 子集 (Subset): 如果集合 AAA 的每个元素都是集合 BBB 的元素，则称 AAA 是 BBB 的子集，记为 A⊆BA \subseteq BA⊆B。
    • 形式化定义：A⊆B  ⟺  ∀x(x∈A→x∈B)A \subseteq B \iff \forall x (x \in A \to x \in B)A⊆B⟺∀x(x∈A→x∈B)。
  • 真子集 (Proper subset): 如果 AAA 是 BBB 的子集，且 A≠BA \neq BA=B，则称 AAA 是 BBB 的真子集，记为 A⊂BA \subset BA⊂B。
    • 形式化定义：A⊂B  ⟺  ∀x(x∈A→x∈B)∧∃x(x∈B∧x∉A)A \subset B \iff \forall x (x \in A \to x \in B) \land \exists x (x \in B \land x \notin A)A⊂B⟺∀x(x∈A→x∈B)∧∃x(x∈B∧x∈/A)。
  • 重要结论：
    • 对于任意集合 SSS，都有 ∅⊆S\emptyset \subseteq S∅⊆S。
    • 对于任意集合 SSS，都有 S⊆SS \subseteq SS⊆S。
    • A=B  ⟺  (A⊆B∧B⊆A)A = B \iff (A \subseteq B \land B \subseteq A)A=B⟺(A⊆B∧B⊆A)。

3. 集合的基数、笛卡尔积与幂集​

  • 基数 (Cardinality): 有限集合 SSS 的基数是其元素的个数，记为 ∣S∣|S|∣S∣。

    • ∣∅∣=0| \emptyset | = 0∣∅∣=0。

  • 容斥原理 (Inclusion-Exclusion Principle): 对于两个有限集合 A,BA, BA,B：

    ∣A∪B∣=∣A∣+∣B∣−∣A∩B∣|\boldsymbol{A \cup B}| = |\boldsymbol{A}| + |\boldsymbol{B}| - |\boldsymbol{A \cap B}|∣A∪B∣=∣A∣+∣B∣−∣A∩B∣

    解释：直接将 ∣A∣|A|∣A∣ 和 ∣B∣|B|∣B∣ 相加时，A∩BA \cap BA∩B 中的元素被计算了两次，因此需要减去一次。

  • n 元组 (n-tuple): 一个有序的元素集合 (a1,a2,...,an)(a_1, a_2, ..., a_n)(a1​,a2​,...,an​)。顺序很重要。

  • 笛卡尔积 (Cartesian Product): 集合 AAA 和 BBB 的笛卡尔积 A×BA \times BA×B 是所有有序对 (a,b)(a, b)(a,b) 的集合，其中 a∈A,b∈Ba \in A, b \in Ba∈A,b∈B。

    • A×B={(a,b)∣a∈A∧b∈B}A \times B = \{ (a, b) \mid a \in A \land b \in B \}A×B={(a,b)∣a∈A∧b∈B}
    • 重要性质:
      • 笛卡尔积不满足交换律，即 A×B≠B×AA \times B \neq B \times AA×B=B×A (除非 A=BA=BA=B 或 A=∅A=\emptysetA=∅ 或 B=∅B=\emptysetB=∅)。

  • 幂集 (Power Set): 集合 SSS 的幂集是 SSS 的所有子集构成的集合，记为 P(S)\mathcal{P}(S)P(S)。

    • 循循善诱的例子:
      1. 如果 S=∅S = \emptysetS=∅，它的唯一子集是 ∅\emptyset∅。所以 P(∅)={∅}\mathcal{P}(\emptyset) = \{\emptyset\}P(∅)={∅}。

      • 解释：构造 SSS 的一个子集时，对于 SSS 中的每一个元素，我们都有两种选择：“放入子集”或“不放入子集”。nnn 个元素，每个都有 2 种选择，根据乘法法则，总共有 2n2^n2n 种组合，即 2n2^n2n 个不同的子集。

4. 函数 (Functions)​

4.1 基本概念​

  • 函数 (Function): 一个从集合 AAA 到集合 BBB 的函数 f:A→Bf: A \to Bf:A→B 是一个规则，它为 AAA 中的每一个元素，都指定了 BBB 中唯一一个对应的元素。
  • 相关术语:
    • 定义域 (Domain): 集合 AAA。
    • 陪域 (Codomain): 集合 BBB。
    • 像 (Image): 如果 f(a)=bf(a) = bf(a)=b，则 bbb 是 aaa 的像。
    • 原像 (Preimage): 如果 f(a)=bf(a) = bf(a)=b，则 aaa 是 bbb 的一个原像。
    • 值域 (Range): 定义域中所有元素的像构成的集合，记为 f(A)f(A)f(A)。值域是陪域的一个子集，即 f(A)⊆Bf(A) \subseteq Bf(A)⊆B。

4.2 函数的类型 (重点)​

类型中文名定义直观理解

Bijective双射 (one-to-one correspondence)既是单射又是满射建立了一个完美的“一一对应”关系。

  • 用例与判断:

    • f:Z→Z,f(x)=x2f: \mathbb{Z} \to \mathbb{Z}, f(x) = x^2f:Z→Z,f(x)=x2。
      • 非单射：因为 f(1)=f(−1)=1f(1) = f(-1) = 1f(1)=f(−1)=1，但 1≠−11 \neq -11=−1。
      • 非满射：因为陪域中没有一个整数的平方等于 −1-1−1。

    • h:R→R,h(x)=2x−1h: \mathbb{R} \to \mathbb{R}, h(x) = 2x-1h:R→R,h(x)=2x−1。

  • 重要定理:

    • 对于从有限集 AAA 到有限集 BBB 的函数 fff，如果 ∣A∣=∣B∣|\boldsymbol{A}| = |\boldsymbol{B}|∣A∣=∣B∣，那么：
    • fff 是单射   ⟺  \iff⟺ fff 是满射。
    • 注意：此定理对无限集不成立。例如，g:Z+→Z+,g(x)=2xg: \mathbb{Z}^+ \to \mathbb{Z}^+, g(x) = 2xg:Z+→Z+,g(x)=2x 是单射但不是满射。

5. 函数的运算与特殊函数​

5.1 逆函数与复合函数​

  • 逆函数 (Inverse Function):

    • 核心条件: 一个函数 f:A→Bf: A \to Bf:A→B 可逆 (invertible) 当且仅当它是一个双射 (bijection)。
    • 逆函数记为 f−1:B→Af^{-1}: B \to Af−1:B→A，定义为 f−1(b)=a  ⟺  f(a)=bf^{-1}(b) = a \iff f(a) = bf−1(b)=a⟺f(a)=b。
    • 为什么必须是双射？
      • 如果不是单射（多对一），那么一个 BBB 中的元素会对应多个 AAA 中的原像，其逆不满足函数的“唯一”性定义。
      • 如果不是满射，那么 BBB 中会有元素在 AAA 中没有对应原像，其逆不满足函数的“每一个”定义域元素都有指定。

  • 复合函数 (Composition of Functions):

    • 设 g:A→Bg: A \to Bg:A→B 和 f:B→Cf: B \to Cf:B→C，则 fff 和 ggg 的复合函数记为 f∘gf \circ gf∘g，定义为 (f∘g)(x)=f(g(x))(f \circ g)(x) = f(g(x))(f∘g)(x)=f(g(x))。
    • 注意：函数复合的顺序是从右到左。
    • 重要性质: 复合运算不满足交换律，即 f∘g≠g∘ff \circ g \neq g \circ ff∘g=g∘f。

  • 向上取整 (Ceiling Function): ⌈x⌉\lceil x \rceil⌈x⌉，表示不小于 xxx 的最小整数。
  • 阶乘 (Factorial Function): n!=n×(n−1)×⋯×2×1n! = n \times (n-1) \times \dots \times 2 \times 1n!=n×(n−1)×⋯×2×1。约定 0!=10! = 10!=1。

7. 无限集的基数 (难点)​

7.1 可数集与不可数集​

  • 基数相等: 两个集合 AAA 和 BBB (可以是无限的) 基数相等，记为 ∣A∣=∣B∣|A|=|B|∣A∣=∣B∣，如果存在一个从 AAA 到 BBB 的双射函数。
  • 可数集 (Countable Set): 一个集合是可数的，如果它是有限的，或者其基数与正整数集 Z+\mathbb{Z}^+Z+ 相同。
    • 直观理解: 可数集中的所有元素都可以被一一列出，形成一个（可能是无限的）列表。
  • 不可数集 (Uncountable Set): 不是可数集的集合。
  • 重要结论:
    • 整数集 Z\mathbb{Z}Z 是可数的。
    • 有理数集 Q\mathbb{Q}Q 是可数的。
    • 任何有限字母表上的所有有限长度字符串的集合是可数的（因此所有 Java 程序的集合也是可数的）。

      • 对于第 iii 位小数 did_idi​，我们让它不等于列表中第 iii 个数字的第 iii 位小数 diid_{ii}dii​。例如，可以取 di=(dii+1)(mod10)d_i = (d_{ii} + 1) \pmod{10}di​=(dii​+1)(mod10)。
    3. 导出矛盾:
      • 这个新构造的数 rrr 也在 [0,1][0, 1][0,1] 区间内。
      • 但是，rrr 不可能等于列表中的任何一个数 rir_iri​，因为它在第 iii 位小数上与 rir_iri​ 不同 (di≠diid_i \neq d_{ii}di​=dii​)。
      • 这与“我们已经列出了所有 [0,1][0, 1][0,1] 内的实数”的假设相矛盾。
    4. 结论: 因此，最初的假设是错误的，[0,1][0, 1][0,1] 区间（以及整个实数集 R\mathbb{R}R）是不可数的。

  • 另一个应用: 同样可以用对角线论证证明自然数的幂集 P(N)\mathcal{P}(\mathbb{N})P(N) 是不可数的。

  • 康托定理 (Cantor's Theorem): 对于任何集合 SSS，其幂集的基数严格大于其自身的基数，即 ∣S∣<∣P(S)∣|S| < |\mathcal{P}(S)|∣S∣<∣P(S)∣。
